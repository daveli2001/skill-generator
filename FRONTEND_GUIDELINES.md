## Overview

## Design Philosophy

Skills created by skill-generator should produce outputs that are:
- **Clean and Professional**: Suitable for business and professional use
- **Readable**: Clear typography and adequate contrast
- **Consistent**: Follow established patterns and conventions
- **Accessible**: Usable by people with disabilities

## Document Outputs

### PDF Documents

- **Page size**: A4 or Letter (based on region)
- **Margins**: 25mm minimum
- **Header**: Company/logo left, page number right
- **Footer**: Document title, date, confidentiality notice

### Excel Spreadsheets

- **Header row**: Bold, background color (Gray 100)
- **Alternating rows**: Light striping for readability
- **Number formatting**: Consistent decimal places
- **Column widths**: Auto-fit to content

### PowerPoint Presentations

- **Slide size**: 16:9 widescreen
- **Title font**: 36pt minimum
- **Body font**: 24pt minimum
- **Contrast**: High contrast for accessibility

---

## Accessibility

### WCAG 2.1 AA Compliance

- **Text contrast**: Minimum 4.5:1 for normal text, 3:1 for large text
- **Color alone**: Never use color as the only means of conveying information
- **Focus states**: Visible focus indicators for interactive elements
- **Alt text**: Descriptive alt text for images and charts

### Screen Reader Support

- Use semantic HTML when generating HTML outputs
- Provide text alternatives for visual content
- Use proper heading hierarchy (H1 → H2 → H3)

## File Format Guidelines

### When to Use Each Format

| Format | Best For |
|--------|----------|
| PDF | Final reports, contracts, official documents |
| DOCX | Editable documents, letters, memos |
| PPTX | Presentations, pitch decks |
| XLSX | Data analysis, financial models, calculations |
| CSV | Data export, importing to other systems |
| PNG | Screenshots, single charts, web images |
| SVG | Logos, icons, scalable graphics |
