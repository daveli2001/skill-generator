# summary-writer

## Overview
This skill helps business professionals quickly summarize long email threads.

## How to Use
Trigger by pasting an email thread and asking for a summary.

## Output Format
- **Key Points:** 3-5 bullet points of main topics
- **Action Items:** List of required actions with owners
- **Timeline:** Important dates mentioned

## Example
**Input:** [Email thread with 10+ messages]
**Output:**
- Key Points: [summary]
- Action Items: [list]
- Timeline: [dates]
